## 🚀 CÓMO USAR

### Ver localmente:
```bash
npm run dev
```
Abre: http://localhost:5173

### Compilar:
```bash
npm run build
```

### Desplegar en Netlify:
1. **Opción rápida**: Arrastra la carpeta `dist` a Netlify
2. **Desde GitHub**: Conecta tu repo y deploy automático
3. **CLI**: `netlify deploy --prod`

## ✏️ PERSONALIZACIÓN FÁCIL

### Cambiar textos:
Cada sección es un archivo separado en `/src/components/`:
- `Concepto.jsx` - Edita filosofía y valores
- `Simbologia.jsx` - Modifica significados
- `Planimetria.jsx` - Ajusta medidas
- `GamaColores.jsx` - Actualiza colores
- `TipografiaSection.jsx` - Cambia fonts
- `Aplicaciones.jsx` - Modifica usos
- `Versiones.jsx` - Ajusta variaciones

### Agregar más logos:
1. Guarda en `/src/components/img/`
2. Actualiza la referencia en el componente
3. Usa ruta: `/src/components/img/TU_IMAGEN.png`
