# Manual de Identidad Visual - Multiaxis SAS

Sitio web interactivo que presenta el manual de identidad corporativa de Multiaxis SAS, desarrollado con React y Bootstrap.

## 📖 Contenido del Manual

### 10 Secciones Incluidas:

1. **Navbar** - Navegación fija con acceso rápido a todas las secciones
2. **Hero** - Portada del manual con logo principal
3. **Concepto** - Filosofía y valores de la marca
4. **Simbología** - Significado de los elementos del logo
5. **Planimetría** - Construcción y proporciones del logotipo
6. **Gama de Colores** - Paleta cromática corporativa completa
7. **Tipografía** - Fuentes oficiales y jerarquía tipográfica
8. **Aplicaciones** - Ejemplos de uso en diferentes materiales
9. **Versiones** - Variaciones del logo para distintos usos
10. **Footer** - Información y recursos adicionales

## 🎨 Sistema de Diseño Implementado

### Paleta de Colores
- **Primario**: #2C5F8D (Azul corporativo)
- **Secundario**: #4A90C7 (Azul claro)
- **Acento**: #F39C12 (Naranja)
- **Oscuro**: #1A1A2E (Azul oscuro casi negro)
- **Gris Medio**: #6C757D
- **Gris Claro**: #E9ECEF

### Tipografía Corporativa
- **Títulos**: Montserrat (300, 400, 500, 600, 700, 800)
- **Texto**: Roboto (300, 400, 500, 700)

## 🚀 Instalación y Uso

### Requisitos Previos
- Node.js 16+ 
- npm o yarn

### Instalación
```bash
npm install
```

### Desarrollo Local
```bash
npm run dev
```

### Compilar para Producción
```bash
npm run build
```

## 🌐 Despliegue en Netlify

1. **Desde GitHub**: Conecta tu repo y deploy automático
2. **Drag & Drop**: Arrastra la carpeta `dist` a Netlify
3. **CLI**: `netlify deploy --prod`

## 📂 Estructura

Todas las imágenes están en `/src/components/img/` y ya están referenciadas en los componentes.

## ✏️ Personalización

Cada sección es un componente independiente en `/src/components/`. Edita el archivo `.jsx` correspondiente para modificar contenido.

## 🛠️ Tecnologías

- React 18.2.0

- Bootstrap 5.3.2
- React Bootstrap 2.9.1
